#介绍
-------------------------------------------
FasterForge是一个专注于简化Minecraft Mod开发的API框架
-------------------------------------------
Minecraft版本:1.12.2
Forge版本:forge:1.12.2-14.23.5.2859
==============================
功能:
    1.使用注解代替Forge原来复杂繁琐的注册方式.
    2.封装了基本的Gui控件.
    3.在未来将会增加粒子特效模块.
==============================
